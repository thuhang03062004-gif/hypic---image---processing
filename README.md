# HYPIC - Digital Image Processing Application 🖼️

## 📖 Project Description
HYPIC is a desktop application developed to perform various digital image processing techniques. This project was created as an academic assignment to apply mathematical and computational algorithms to real-world image data.

## 🛠️ Technologies & Tools
* **Language/Platform:** MATLAB
* **UI Framework:** MATLAB App Designer

## ⚙️ Key Features & Techniques
* **Image Thresholding**
* **Histogram Equalization**
* **Noise Filtering**
* **User-friendly Interface:** Allows users to easily upload images, apply filters, and view visual comparisons.

## 🏆 Key Results
* Successfully developed a fully functional application interface.
* Achieved a perfect score of **10.0/10.0** in the *Image Processing with Matlab* course.

## 🚀 How to Run
1. Extract the `MATLAB.zip` file.
2. Open MATLAB.
3. Run the main `HYPIC.mlapp` file to launch the HYPIC interface.
